# Identifying Undervalued Players 2025-2026

**Data Science Portfolio Project**  
Top 5 European Leagues • FBref Data • Clustering + XGBoost

**Live Dashboard** → [Link will go here later]

## Project Overview
End-to-end analysis to find undervalued football players using performance data from the 2025-2026 season.

## Key Features
- Player role clustering (K-Means)
- Market value prediction (XGBoost)
- Interactive Streamlit dashboard
- Full EDA and feature engineering

## Repository Structure
- `notebooks/` → All analysis notebooks
- `dashboard/` → Interactive web app
- `visuals/` → Charts and screenshots
- `data/` → Dataset (not committed)

## Technologies
Python, Pandas, Scikit-learn, XGBoost, Plotly, Streamlit
